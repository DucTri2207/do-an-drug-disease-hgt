## Lỗi phát sinh và cách xử lý

1. **Lỗi `streamlit` command not recognized**
   - Khắc phục bằng `python -m streamlit run ...` và cài `streamlit` trong venv.

2. **Lỗi load checkpoint do config key cũ (`hidden_dims`)**
   - Khắc phục bằng backward-compat mapping trong `src/inference.py`.

3. **Lỗi SyntaxError trong `trainer.py` (dính 2 lệnh cùng dòng)**
   - Khắc phục bằng sửa tách dòng đúng cú pháp.
