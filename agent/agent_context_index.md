# Agent Context Index

Tệp này gom tất cả các file hướng dẫn / ngữ cảnh `.md` quan trọng để agent và người dùng tra cứu nhanh.

## Danh sách file ngữ cảnh

1. `AGENTS.md`
   - Quy ước làm việc chính cho agent trong repo.

2. `ai_agent_context_hgt_drug_protein_disease.md`
   - Ngữ cảnh học thuật và kỹ thuật đầy đủ cho đề tài HGT Drug–Protein–Disease.

3. `README.md`
   - Tổng quan dự án, cách chạy, cấu trúc.

4. `plans/master-plan.md`
   - Kế hoạch tổng thể theo pha/sprint.

5. `plans/engineering-plan-amdgt-reuse.md`
   - Kế hoạch triển khai chuẩn kỹ sư, mapping và chiến lược reuse từ AMDGT.

## Ghi chú sử dụng

- Khi bắt đầu phiên làm việc mới, ưu tiên đọc theo thứ tự:
  1) `AGENTS.md`
  2) `ai_agent_context_hgt_drug_protein_disease.md`
  3) Các file trong `plans/`
  4) `README.md`
- Nếu thêm file ngữ cảnh `.md` mới, hãy cập nhật index này ngay.
